This project is a Python-based backend evaluation system designed to test and analyze MCP servers using an Anthropic Claude-powered AI agent. It organizes complex question-answer evaluations against MCP servers to measure their effectiveness and provides tooling to manage MCP connections and run these evaluations asynchronously.

---
The project is a Python backend evaluation framework for MCP servers, using Anthropic's Claude model to run complex, multi-step test questions. It manages asynchronous MCP connections and tool calls to assess the server's ability to handle realistic, multi-tool workflows.

A static project overview page has been created as public/index.html, detailing the backend's purpose, key components, dependencies, and usage instructions for a local run. There is no frontend UI, so this static overview is the best representation for the preview. Open the Preview tab to view it.
