🧠 How the “local Claude” setup works

Claude Code = the agent shell
Ollama = the local model engine
Qwen / Llama / Mistral = the local brain

Claude Code thinks it’s talking to Anthropic, but you redirect it to your local model.

🛠️ Install Claude‑style AI locally (server or PC)

Below is the real working method from the sources.

1️⃣ Install Ollama (local LLM engine)

• Linux: curl -fsSL https://ollama.com/install.sh | sh
• Windows: irm https://ollama.com/install.ps1 | iex
DEV Community


2️⃣ Install Claude Code

• npm install -g @anthropic-ai/claude-code
DEV Community


3️⃣ Pull a local model

Examples:

• ollama pull qwen2.5-coder:7b
• ollama pull qwen3-coder:30b (needs 32GB+ RAM)
LinkedIn


4️⃣ Point Claude Code to your local model

Set environment variables:

export ANTHROPIC_BASE_URL=http://localhost:11434/v1
export ANTHROPIC_AUTH_TOKEN=ollama
export CLAUDE_CODE_DISABLE_NONESSENTIAL_TRAFFIC=1


LinkedIn

5️⃣ Launch Claude Code

claude


Now it uses your local model, not Anthropic’s cloud.
